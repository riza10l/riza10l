## Ringkasan Eksekutif dan Filosofi Rekayasa

Mahasiswa Ilmu Komputer di Universitas Pendidikan Indonesia yang memiliki minat pada kecerdasan buatan, machine learning, rekayasa sistem, dan analisis kuantitatif. Fokus utama terletak pada pengembangan arsitektur perangkat lunak yang andal, optimalisasi pipeline data, serta pembangunan sistem cerdas yang efisien dan dapat dikembangkan secara berkelanjutan.

Pendekatan rekayasa yang diterapkan menitikberatkan pada kualitas implementasi, konsistensi data, desain kode yang bersih, serta efisiensi pemanfaatan sumber daya komputasi. Setiap proyek dirancang dengan tujuan menghasilkan solusi yang terukur, terdokumentasi dengan baik, dan memiliki dampak nyata terhadap permasalahan yang diselesaikan.

### Bidang Keahlian

* **Infrastruktur AI dan Computer Vision:** Pengembangan pipeline machine learning yang dapat direproduksi, eksplorasi data terstruktur, serta implementasi sistem pemrosesan citra berbasis OpenCV.
* **Sistem Tertanam dan Rekayasa Perangkat Keras:** Pengembangan purwarupa berbasis mikrokontroler, eksperimen optimisasi perangkat keras, dan eksplorasi sistem komputasi khusus.
* **Analisis Kuantitatif dan Rekayasa Data:** Pembangunan sistem pemrosesan data temporal untuk pemodelan pasar, simulasi matematis, dan analisis performa berbasis data.

### Kesempatan Kolaborasi

* Kolaborasi riset pada bidang kecerdasan buatan, machine learning, dan komputasi terdistribusi.
* Program magang dan proyek profesional yang berfokus pada backend engineering, rekayasa data, dan pengembangan sistem cerdas.
* Kontribusi terhadap proyek open source yang berkaitan dengan optimisasi komputasi dan pengembangan teknologi inovatif.

<br />

### Deskripsi Proyek

Framework machine learning dan deep reinforcement learning yang dirancang untuk memodelkan keputusan alokasi aset secara adaptif, mengoptimalkan rasio risiko dan keuntungan, serta meminimalkan drawdown pada data finansial berbasis runtun waktu.

| Aspek         | Detail                                                                               |
| :------------ | :----------------------------------------------------------------------------------- |
| Teknologi     | Python, PyTorch, Stable-Baselines3, Pandas, NumPy, Scikit-Learn                      |
| Skala Sistem  | Mendukung simulasi data runtun waktu dengan frekuensi tinggi                         |
| Performa      | Inferensi model dioptimalkan untuk menghasilkan keputusan secara efisien             |
| Validasi Data | Menggunakan pemisahan data yang ketat untuk mencegah look-ahead bias                 |
| Hasil         | Menunjukkan peningkatan efisiensi strategi alokasi dinamis dibandingkan metode dasar |
| Repositori    | `riza10l/MLRL01`                                                                     |

### Gambaran Arsitektur

MLRL01 dikembangkan untuk menghadapi dinamika pasar yang kompleks dan berubah dengan cepat. Agen cerdas dilatih menggunakan lingkungan kustom berbasis Gym untuk mempelajari strategi alokasi aset yang optimal melalui mekanisme reward yang mempertimbangkan risiko. Pemanfaatan struktur data multidimensi memungkinkan proses komputasi berjalan secara efisien.
